# 前言

欢迎来到基于SSM的软件项目管理系统！此项目致力于为广大开发者提供一套高效、实用的项目管理工具，通过运用Spring、Spring MVC和MyBatis等主流技术，实现项目管理各环节的便捷操作。以下是本项目的详细介绍。

# 内容介绍

本项目是一款基于Java语言的软件项目管理平台，采用前后端分离的设计模式，前端使用Vue.js框架，后端基于SSM框架（Spring、Spring MVC和MyBatis）。主要功能包括项目创建、任务分配、进度跟踪、风险评估等，满足企业在软件开发过程中的项目管理需求。通过本项目，可以显著提高开发团队的工作效率，降低项目风险。

# 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
